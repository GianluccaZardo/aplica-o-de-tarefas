# To-Do List React (Simples com HTML + JS + CSS)

Este projeto é uma aplicação **To-Do List** desenvolvida com **React**, utilizando:

- Rotas com **React Router**
- Gerenciamento global de estado com **Context API**
- Estado e efeitos com **Hooks (useState, useEffect)**
- Persistência de dados com **localStorage**
- Estrutura simples: **HTML + CSS + um único arquivo JS**
- Compatível com **Live Server (Go Live)** usando **HashRouter**

Este projeto foi feito para estudo e prática de React sem usar bundlers ou Create React App.

---

## Funcionalidades

 Adicionar tarefas  
 Listar tarefas  
 Editar tarefas  
 Excluir tarefas  
 Dados salvos no localStorage  
 Navegação entre páginas com React Router  
 Estado global com Context API  
 Totalmente funcional no Live Server
